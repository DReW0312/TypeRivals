# ⌨️ TypeRivals

TypeRivals is a fast, minimalistic, and fully interactive typing speed test application. It allows users to test their typing speed, track their accuracy across different time intervals, and save their scores to a personal profile.

## ✨ Features

* **Real-time Metrics:** Live calculation of Words Per Minute (WPM) and typing accuracy.
* **Custom Timers:** Choose between 30-second, 60-second, or 120-second typing tests.
* **User Authentication:** Secure email and password sign-up/login system.
* **User Profiles:** Customize your account by uploading a profile picture.
* **Score History:** Automatically saves your past contest results (WPM, Accuracy, and Date) to a database so you can track your improvement over time.
* **Sleek UI:** A clean, distraction-free dark mode interface.

## 🛠️ Tech Stack

This project was built using a lightweight, vanilla front-end and a modern cloud back-end:

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Backend / Database:** [Firebase Firestore](https://firebase.google.com/docs/firestore) (NoSQL database for user data and history)
* **Authentication:** [Firebase Auth](https://firebase.google.com/docs/auth)
* **Image Hosting:** [Cloudinary](https://cloudinary.com/) API (Unsigned image uploads for profile pictures)
* **Hosting:** [GitHub Pages](https://pages.github.com/)
